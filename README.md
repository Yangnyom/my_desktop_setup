## ⭐ 3. 핵심 기능
> 프론트엔드 환경에서 3주 내에 구현할 수 있는 핵심적인 기능 딱 3가지를 정의합니다.

1. **[기능명 1 : 다양한 디자인의 제품 드래그 앤 드롭]**: 마우스, 키보드, 본체, 데스크 등 다양한 디자인의 제품들을 드래그앤 드롭하며 원하는 제품을 배치할 수 있음.
2. **[기능명 2 : 자유로운 배치]**: 다양한 제품들을 내가 원하는 위치에 배치해 내 마음에 드는 최적의 데스크셋업 배치를 할 수 있음.
3. **[기능명 3 : 시간대별 책상 배경]**: 시간대별로(낮, 밤) 배경 색상을 어둡게 또는 밝게 설정해 조명의 여부도 확인할 수 있음.

## 🗺️ 4. 화면 설계 및 유저 플로우

* 사용자가 우리 웹사이트에 접속하여 목표를 달성하기까지의 클릭 흐름을 단계별로 작성합니다.
  1. 메인 페이지 접속
  2. "셋업 시작하기" 버튼 클릭 -> 에디터 화면으로 진입
  3. 왼쪽 사이드바에서 카테고리 선택 (책상, 모니터, 키보드 등)
  4. 원하는 아이템을 3D 뷰포트로 드래그 앤 드롭하여 배치
  5. 아이템 클릭 후 위치·회전 조정
  6. 마음에 안 드는 아이템 삭제 또는 되돌리기
  7. 완성된 셋업 스크린샷으로 저장
